# Tienda React - Estilos + Git Profesional

Aplicación de catálogo de productos hecha en React con Vite, enfocada en estilos globales coherentes y buenas prácticas de Git.

## Descripción
Catálogo responsive que muestra productos en cards con hover effects, estados de stock y categorías. El proyecto usa un sistema de variables CSS en `:root` para mantener consistencia de colores y espaciados en toda la app.

## Tecnologías
- **React 18** - Librería para UI
- **Vite** - Build tool y dev server
- **CSS Puro** - Variables CSS, Flexbox, Grid, Media Queries
- **JavaScript ES6+**

## Características
- Sistema de diseño con variables CSS en `:root`
- Reset universal y estilos base coherentes
- Cards con sombra, border-radius y efecto hover
- Grid responsive: 2 columnas en desktop, 1 en mobile
- Validación visual de formularios con `:focus`
- Fondo con degradado aplicado al contenedor principal
- Componentes reutilizables: `Card.jsx`
